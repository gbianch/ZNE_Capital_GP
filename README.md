# Identifying favorable regions for investing in rooftop PV on multifamily housing  

Reproducible code for master's group project by students in the Bren School at the University of California Santa Barbara. 


This code reads and wrangles data involving population size, employment rates, median household income, median house price, ... Metrics, such as population change and change in house prices, are calculated from data to provide normalized values to compare MSAs. Final demographic data is not tidy for the purposes of utlizing in ArcGIS Pro.
