# Finding suitable Metropolitan Statistical Areas(MSAs) in the U.S. for ZNE Capital to buy real estate using demographic trends
Reproducible code for master's group project by students in the Bren School at the University of California Santa Barbara. Preliminary analysis of MSA demographic trends to select regions with "significant" population growth since 2020.

This code reads and wrangles data involving population size, employment rates, median household income, median house price, ... Metrics, such as population change and change in house prices, are calculated from data to provide normalized values to compare MSAs. Final demographic data is not tidy for the purposes of utlizing in ArcGIS Pro.

Historical data is analyzed in a separate .Rmd document. This analysis is to compare suitable MSAs once narrowed down using utility data.
